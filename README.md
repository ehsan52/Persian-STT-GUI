# Persian STT GUI

A lightweight graphical interface for Persian speech-to-text, powered by the **Persian-STT model**  
created by the **Oct4Pie** project.  
The GUI design and graphics are created by **Ehsan Eskandari (ehsan52)**.

---

## Features
- Persian speech recognition using the Coqui STT engine  
- Simple Tkinter-based graphical interface  
- Load audio files or record directly from microphone  
- Display recognized text in the application

---

## Dependencies
- Python 3.12  
- Coqui STT  
- TensorFlow Lite Runtime  
- Tkinter  

---
